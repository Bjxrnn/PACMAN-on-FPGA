# PACMAN-on-FPGA
A Pac-Man-inspired maze game, displayed on a 96×64 OLED screen using a 19×12 tile grid. The system features a randomised maze each round, pellet system, 3-lives system with immunity, three unique ghost AI behaviours selectable across three difficulty levels, power pellets with unique effects, a live score display, and in-game audio from PmodAMP2.
